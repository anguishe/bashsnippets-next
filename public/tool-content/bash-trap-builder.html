05Signal reference

    

      
        
SignalWhen it firesCommon use case

      
      
        
EXIT (0)Script exits for any reason — success, failure, or interrupt.The one trap that always runs. Put resource teardown here.

        
ERRA command returns a non-zero exit code (only while set -e is on).Log the exact failing line with $LINENO before the script dies.

        
INT (2)User presses Ctrl+C at the terminal.Print an interrupted message and exit 130 so callers see it was cancelled.

        
TERM (15)kill, systemctl stop, or docker stop sends SIGTERM.Graceful shutdown of long-running scripts and Docker entrypoints.

        
HUP (1)Terminal closes or the SSH session drops.Decide whether to abort or keep running when the session is lost.

        
PIPEWriting to a pipe whose reader has closed (e.g. head exited early).Suppress broken-pipe noise; usually handled rather than aborted on.
